https://api.covid19api.com/summary

---
How to download and use this application:

- Clone this repository
```bash
  git clone https://github.com/diegoveigass/covid-19
```
- Go to folder
```bash
  cd covid-19
```
- Install the dependencies
```bash
  yarn
```
- Start the application
```bash
  yarn start
```
